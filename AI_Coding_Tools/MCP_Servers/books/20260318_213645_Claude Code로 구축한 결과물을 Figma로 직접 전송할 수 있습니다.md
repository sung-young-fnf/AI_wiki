### Claude (팔로워 761,428명)
3주 전 • 웹에서 공개

이제 Claude Code에서 구축 중인 결과물을 Figma로 직접 전송할 수 있습니다.

Figma의 최신 MCP 서버(MCP server) 버전을 사용하면 Claude Code로 제작한 작동 중인 프로토타입(prototype)을 Figma 캔버스(canvas)로 보낼 수 있습니다. 코드를 분기(branching)하거나 처음부터 다시 구축할 필요 없이 다양한 디자인 방향을 나란히 탐색할 수 있습니다.

**시작하려면:**
`/plugin install figma@claude-plugin-directory`

그런 다음 Claude에게 로컬 웹 앱(web app)의 페이지를 Figma로 보내도록 프롬프트(prompt)를 입력하세요.

원격 MCP 서버(remote MCP servers)에 Claude Code를 연결하는 방법에 대한 자세한 내용은 다음 문서를 참조하세요: https://lnkd.in/eWnNn7kM
