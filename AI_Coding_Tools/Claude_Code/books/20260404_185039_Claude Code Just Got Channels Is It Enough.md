# Claude Code Just Got Channels. Is It Enough?

앤스로픽(Anthropic)이 클로드 코드(Claude Code)에 새로운 디스코드(Discord) 연동 기능을 출시했습니다. 필자는 스마트폰으로 코딩하기 위해 클로드 코드와 4가지 오픈소스 대안을 테스트해 보았습니다. 격차는 줄어들고 있지만, 아직 완벽하게 해소되지는 않았습니다.

### '책상세(The Desk Tax)'의 문제

필자는 클로드 코드에 매달 20달러를 지불합니다. 하지만 이 도구는 책상 앞에 앉아 있는 하루 2~3시간 동안만 사용될 뿐, 출퇴근 시간이나 점심시간 등 나머지 시간 동안 구독 서비스는 유휴 상태로 방치됩니다. 이를 필자는 '책상세(The Desk Tax)'라고 부릅니다. 터미널 앞에 앉아 있을 때만 작동하는 AI 도구의 숨겨진 비용인 셈입니다.

이번 주, 앤스로픽은 이 비용을 줄여줄 두 가지 기능을 출시했습니다. 하나는 동시 세션(concurrent sessions)과 깃 워크트리 격리(git worktree isolation)를 지원하는 **리모트 컨트롤(Remote Control)** 서버 모드이고, 다른 하나는 텔레그램(Telegram)이나 디스코드를 통해 클로드 코드 세션에 메시지를 보낼 수 있는 **채널(Channels)** 기능입니다.

### 도구별 비교 지형도

모바일에서 클로드 코드와 코덱스(Codex)를 실행하는 5가지 방법을 비교했습니다.

1.  **공식 리모트 컨트롤 & 채널(Official Remote Control + Channels):** 가장 설정이 빠르며 앤스로픽 API를 통해 라우팅됩니다. 
2.  **Happy Coder (오픈소스):** iOS 및 안드로이드 네이티브 앱을 제공하며 음성 제어가 강점입니다.
3.  **HAPI (오픈소스):** 클로드 코드뿐만 아니라 코덱스, 제미나이(Gemini) 등 여러 에이전트를 지원하며 텔레그램 연동이 강력합니다.
4.  **Tailscale + SSH + tmux (DIY):** 데이터가 자신의 네트워크를 벗어나지 않는 제로 트러스트(zero-trust) 방식입니다. 보안이 가장 뛰어나지만 설정이 번거롭습니다.
5.  **Myrlin Workbook (오픈소스):** 워크스페이스 관리 도구로, 웹 UI를 통해 모바일 접근이 가능합니다.

### 주요 비교 포인트

*   **제어 및 시작 속도:** 리모트 컨트롤은 설정이 가장 빠르지만, 여전히 데스크톱에서 세션을 먼저 시작해야 합니다. 반면 HAPI는 모바일에서 직접 세션을 생성하고 에이전트를 전환할 수 있습니다.
*   **모바일 경험:** Happy Coder와 HAPI는 음성 제어를 지원합니다. 공식 채널 기능은 텔레그램 알림을 통해 클로드가 입력을 기다릴 때 알려주므로 모니터링 부담을 줄여줍니다.
*   **연결 안정성:** WireGuard 기반의 HAPI와 Tailscale 방식이 가장 회복력이 좋습니다. 공식 리모트 컨트롤은 네트워크가 10분 이상 끊기면 세션이 종료되는 한계가 있습니다.

### 공식 경로: 리모트 컨트롤과 채널

리모트 컨트롤은 클로드 코드에 내장되어 있어 별도의 도구가 필요 없습니다. 터미널에서 `claude remote-control`을 실행하고 QR 코드를 스캔하면 모바일 앱에서 제어할 수 있습니다.

새로 추가된 **채널(Channels)** 기능은 리모트 컨트롤에 부족했던 푸시 알림(push notifications) 레이어를 보완합니다. 텔레그램 봇을 설정하면, 클로드가 작업을 완료하거나 질문이 있을 때 텔레그램으로 메시지를 보냅니다. 사용자는 텔레그램에서 바로 답장하여 작업을 이어갈 수 있습니다.

**설정 방법 요약:**
1.  클로드 코드를 최신 버전(v2.1.80 이상)으로 업데이트합니다.
2.  `claude remote-control` 명령어로 서버 모드를 실행합니다.
3.  텔레그램 플러그인을 설치하고 봇 토큰을 설정하여 연동합니다.

### 파워 유저를 위한 선택: HAPI

HAPI는 공식 리모트 컨트롤이 해결하지 못한 '모바일에서의 세션 시작' 문제를 해결합니다.
*   **원격 시작:** 데스크톱 근처에 없어도 스마트폰 앱(PWA)에서 직접 세션을 생성할 수 있습니다.
*   **멀티 에이전트:** 클로드 코드 외에도 여러 AI 에이전트를 한 곳에서 관리합니다.
*   **보안 옵션:** 기본 릴레이 외에도 Cloudflare 터널이나 Tailscale을 통해 보안을 강화할 수 있습니다.

### 대안들의 특징

*   **Happy Coder:** 세련된 네이티브 앱 UI와 강력한 음성 제어 기능(voice-to-action)이 특징입니다. 이동 중에 코딩 명령을 내리기에 가장 적합합니다.
*   **Tailscale + SSH:** 보안 규정이 엄격한 환경에 적합합니다. 서드파티 서버를 전혀 거치지 않지만, 6인치 화면에서 터미널 명령어를 직접 입력해야 하는 불편함이 있습니다.

### 결론: 당신에게 맞는 도구는?

*   **입문자:** 설정이 간편한 **공식 리모트 컨트롤과 채널** 조합을 추천합니다. 알림 기능만으로도 생산성이 크게 향상됩니다.
*   **고급 사용자 및 멀티 에이전트 사용자:** 모바일에서 직접 세션을 제어할 수 있는 **HAPI**가 유리합니다.
*   **보안 중시 및 기업용:** 데이터 유출 방지를 위해 **Tailscale + SSH** 조합이 유일한 해답이 될 수 있습니다.

클로드 코드의 채널 기능은 '책상세'를 줄이는 데 큰 역할을 했지만, 여전히 모바일에서 세션을 직접 시작할 수 없다는 한계가 있습니다. 기술의 발전 속도로 볼 때, 머지않아 책상 앞이 아니더라도 완벽한 코딩 환경을 구축할 수 있게 될 것입니다.
