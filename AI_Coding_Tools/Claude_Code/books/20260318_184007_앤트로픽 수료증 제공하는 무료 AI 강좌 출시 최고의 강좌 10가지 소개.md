### 1. Claude Code 활용하기 (Claude Code in Action)

이 강좌는 개발 속도를 높이고 싶은 개발자를 위한 것입니다. 실제 개발 워크플로(development workflow)를 가속화하기 위해 Claude Code를 활용하는 방법을 다룹니다. 장난감 같은 예시가 아닌 실제 코딩 작업에 중점을 둡니다.

이미 코드를 작성하고 있으며 AI가 더 많은 수고를 덜어주기를 원한다면 이 강좌부터 시작하십시오.

### 2. Claude 101 (Claude 101)

기본 입문 강좌입니다. Claude를 일상적인 생산성 향상에 활용하는 기초를 다룹니다. 효과적인 프롬프트 작성법, 유용한 결과물 얻는 법, 그리고 일상 업무에서 AI를 사용하는 좋은 습관을 기르는 방법을 배웁니다.

Claude를 처음 접하거나 간단한 질문 외에 무엇을 할 수 있는지 제대로 탐색해보지 않은 모든 사람에게 적합합니다.

[https://anthropic.skilljar.com/claude-101](https://anthropic.skilljar.com/claude-101)

### 3. AI 활용 능력: 프레임워크 및 기본 원칙 (AI Fluency: Framework & Foundations)

이 강좌는 기술적인 강좌와는 다릅니다. AI와 효과적이고 안전하며 윤리적으로 협업하는 방법에 중점을 둡니다. 코드보다는 판단력에 대한 내용이 많습니다.

기술 전문가라도 수강할 가치가 있습니다. 많은 개발자가 'AI에 대해 어떻게 생각해야 하는가'라는 측면을 간과하며, 이는 그들의 작업에서 나타납니다.

[https://anthropic.skilljar.com/ai-fluency-framework-foundations](https://anthropic.skilljar.com/ai-fluency-framework-foundations)

### 4. Claude API로 개발하기 (Building with the Claude API)

Claude 기반으로 애플리케이션을 구축하고 싶다면 이 강좌가 적합합니다. API, 모델을 프로그래밍 방식으로 다루는 방법, 그리고 요청(request)을 올바르게 구성하는 방법을 다룹니다.

작업할 내용 미리 보기:

```python
import anthropic

client = anthropic.Anthropic()
message = client.messages.create(
    model="claude-opus-4-5",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Hello, Claude"}
    ]
)
print(message.content)
```

이것이 기본적인 API 호출(API call) 형태입니다. 강좌는 오류 처리(error handling), 스트리밍(streaming), 도구 사용(tool use), 그리고 이를 활용한 실제 애플리케이션 구축 등 훨씬 더 심층적인 내용을 다룹니다.

[https://anthropic.skilljar.com/claude-with-the-anthropic-api](https://anthropic.skilljar.com/claude-with-the-anthropic-api)

### 5. 모델 컨텍스트 프로토콜 소개 (Introduction to Model Context Protocol)

MCP(Model Context Protocol)는 Claude를 외부 도구, 데이터베이스, 서비스에 연결하는 방식입니다. 지난 몇 달간 에이전트(agent) 분야에서 매우 중요해졌습니다.

이 강좌는 MCP가 실제로 무엇인지, 어떻게 작동하는지, 그리고 Claude의 기능을 확장하기 위해 어떻게 사용하는지 설명합니다. 이 용어를 자주 접했지만 제대로 이해하고 싶다면 이 강좌가 제격입니다.

[https://anthropic.skilljar.com/introduction-to-model-context-protocol](https://anthropic.skilljar.com/introduction-to-model-context-protocol)

### 6. 교육자를 위한 AI 활용 능력 (AI Fluency for Educators)

교사를 위해 특별히 설계되었습니다. 교육적 맥락에서 AI를 어떻게 생각해야 하는지, 학생들에게 책임감 있게 사용하는 방법, 그리고 실제로 도움이 되는 방식으로 AI를 교수법에 통합하는 방법을 다룹니다.

교육 분야에 종사하며 AI와의 관계를 어떻게 정립해야 할지 고민하고 있다면, 이 강좌는 바로 당신을 위해 만들어졌습니다.

### 8. 에이전트 스킬 소개 (Introduction to Agent Skills)

Claude Code에는 '스킬(skills)'이라는 개념이 있습니다. 이는 에이전트(agent)에게 특정 작업을 처리하는 방법을 알려주는 재사용 가능한 명령어 세트입니다. 이 강좌는 스킬을 구축하는 방법을 가르쳐줍니다.

Claude Code를 정기적으로 사용하며 동일한 컨텍스트(context)를 반복해서 작성하는 것을 멈추고 싶다면, 에이전트 스킬이 해답입니다. 이 강좌는 스킬을 올바르게 작성하는 방법을 보여줍니다.

[https://anthropic.skilljar.com/introduction-to-agent-skills](https://anthropic.skilljar.com/introduction-to-agent-skills)

### 어디서부터 시작해야 할까요?

어떤 역할인지에 따라 다릅니다.

*   **AI가 처음이라면:** Claude 101, 이어서 AI 활용 능력: 프레임워크 및 기본 원칙.
*   **개발자라면:** Claude API로 개발하기, 이어서 모델 컨텍스트 프로토콜 소개.
*   **Claude Code를 사용한다면:** Claude Code 활용하기, 이어서 에이전트 스킬 소개.
*   **교육 분야에 종사한다면:** 역할에 따라 교육자 또는 학생 전용 강좌.
