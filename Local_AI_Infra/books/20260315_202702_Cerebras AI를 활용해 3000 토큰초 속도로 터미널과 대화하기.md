# Cerebras AI를 활용해 3,000 토큰/초 속도로 터미널과 대화하기

최신 소식에 귀 기울이셨다면, 아마 Cerebras에 대해 들어보셨을 겁니다. 간단히 말해, 이들은 막대한 웨이퍼 스케일 엔진(Wafer-Scale Engine)을 활용하여 AI 응답(이른바 추론, inference)을 놀라운 속도로 전달하도록 설계된 고성능 추론 서비스(inference service)를 제공합니다.

이들의 서비스는 개발자들이 Llama, Qwen, GLM, GPT와 같은 오픈소스 모델을 기존 엔비디아(Nvidia) GPU 기반 클라우드보다 최대 20배 빠른 속도로 실행할 수 있도록 지원합니다.

이는 매우 중요한 의미를 가집니다.

최근에는 오픈AI(OpenAI)조차 이 동일한 플랫폼에서 강력한 코덱스(Codex) 모델을 출시하기 시작했으며, 초당 1,000개 이상의 토큰(tokens) 추론 속도를 자랑합니다 — 그야말로 엄청난 속도입니다!

## GPT-5.3-Codex-Spark 소개

GPT-5.3-Codex-Spark를 소개합니다 — 우리의 첫 실시간 코딩 모델입니다. 15배 더 빠른 생성 속도, 128k 컨텍스트, 현재 연구 중…

openai.com

아쉽게도, 월 200달러가 넘는 프로(Pro) 플랜이 없으면 코덱스 스파크(Codex Spark)를 테스트할 수 없습니다.

이 기술이 얼마나 혁신적인지 보여드리기 위해, 자연어로 터미널과 대화하고 즉각적인 응답과 명령을 받는 방법을 시연해 보이겠습니다. 물론, 이 모든 것은 완전히 무료입니다.

## 이 예시를 선택한 이유

터미널에서 복잡한 명령을 실행해야 하는데, 정확한 명령어나 명령어 시퀀스가 기억나지 않는다고 상상해 보세요. 수십 개의 매우 똑똑한 모델들에게 물어볼 수 있고, 그들은 아마도 당신을 도울 것입니다.

하지만 한 가지 문제가 있습니다. 바로 응답 시간입니다. 때로는 작업 흐름(flow)에 몰입해 있을 때, AI가 생각하고 추론하기 위해 몇 초를 기다리는 것이 집중력을 깨뜨릴 수 있습니다. 초고속 추론(inference)을 사용하면, 복잡한 명령도 생각보다 빠르게 생성될 수 있습니다. 당신이 할 일은 그저 확인하고 실행하는 것뿐입니다.

## 결과는 다음과 같습니다:

*전체 크기로 이미지를 보려면 엔터 키를 누르거나 클릭하세요.*

참고로 이 영상은 어떠한 가속도 없이 촬영되었다는 점에 유의하세요. 응답 속도는 Cerebras에서 제공하는 그대로이며, 초당 최대 3,000개 토큰에 달합니다. 이는 우리가 거의 즉각적인 응답을 얻을 수 있게 해 주며, 이는 외운 명령을 직접 타이핑하는 것보다도 더 빠릅니다.

Cerebras GPT-OSS 120을 사용하여 자연어로 터미널 명령을 실행하기 위해 필요한 것은 다음과 같습니다:

*   cerebras.ai에서 무료 Cerebras 계정 생성
*   셸 GPT(Shell GPT) 설치 (https://github.com/tbckr/sgpt)

다음의 간단한 단계를 따르세요. Mac의 경우 아래 단계를 따르시면 되고, Windows의 경우 SGPT 문서에서 경로를 확인해 주세요.

### 1단계: 설치

pip 또는 문서에 설명된 다른 방법 중 하나를 사용하여 설치합니다.

```bash
pipx install shell-gpt
```

여기에서 Cerebras용 무료 API 키를 받으세요.

https://cloud.cerebras.ai/

### 2단계: 구성 (Configure)

그다음 SGPT 구성 파일(configuration file)을 엽니다.

일반적으로 다음 경로에 있습니다:

```
~/.config/shell_gpt/.sgptrc_
OPENAI_API_KEY=<YOUR CEREBRAS KEY HERE>
API_BASE_URL=https://api.cerebras.ai/v1
DEFAULT_MODEL=gpt-oss-120b
```

### (선택 사항) 3단계: 별칭(Alias) 추가

예를 들어 'sgpt show me all images'를 입력하는 것처럼 프롬프트(prompt)를 터미널에 직접 전달하려면, 아래 줄을 셸 구성 파일(bashrc, zsh 등)에 추가하세요.

```bash
ai() {
  ~/.local/bin/sgpt -s "$*"
}
```

이제 소개 영상에서 시연된 방법을 사용하여 테스트할 수 있습니다.

```bash
ai count all lines in python files
```

몇 초 후 다음과 같은 내용이 표시될 것입니다. 엔터를 누르면 명령을 확장할 수 있음을 확인합니다.

아니면 다음과 같은 것을 시도해 볼 수도 있습니다:

```bash
ai move all fiels with screenshot in the filename to a new folder
```

아니면 떠오르는 어떤 것이든 시도해 보세요!

실행 전에 확인만 하면 되는 즉각적인 응답을 받을 것입니다 (명령을 확인하기 전에 반드시 검토하시길 강력히 권장합니다).
