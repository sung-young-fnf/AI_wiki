# Cotypist 맥 앱 심층 리서치 및 설치 방법

> 조사일: 2026-04-19
> 대상: Cotypist — macOS AI 자동완성 앱

---

## 1. 개요
**Cotypist**는 독일 인디 개발자 **Daniel Gräfe**가 만든 macOS 전용 **AI 자동완성(AI Autocomplete)** 앱. 모바일처럼 거의 모든 Mac 앱의 텍스트 필드에서 다음 단어/구문을 실시간으로 예측·제안. 온디바이스 LLM으로 작동해 **모든 처리가 로컬에서 이뤄지는 프라이버시 중심** 설계가 핵심 차별점.

## 2. 핵심 기능
- **범용 인라인 자동완성**: Safari, Chrome, Apple Mail, MS Word, TextEdit, Notes, Obsidian, Spark 등 대부분 Mac 앱의 텍스트 필드에 자연스럽게 통합. Tab으로 수락, Esc로 무시.
- **온디바이스 로컬 LLM**: Qwen 2.5, Llama 3.2, Gemma 2 등 로컬 모델 지원. 사용자 문장이 클라우드로 전송되지 않음.
- **학습 기능**: 사용자가 수락한 제안을 학습해 문체에 점차 적응 (데이터는 로컬 저장).
- **터미널 스마트 동작**: iTerm/Terminal에서는 AI 프롬프트 입력 시에만 자동 활성화 (셸 자동완성 방해 회피).
- **시스템 설정 존중**: macOS Smart Quotes, 앱별 제외 목록, 단축키 제외 설정 지원.
- **사용 효과 추적**: 절약된 타이핑 시간 통계 제공.

## 3. 제약 / 호환성 이슈
- **미지원/제한 앱**: Pages, Scrivener, Thunderbird, 주요 코드 에디터 본문 (VS Code는 사이드바·AI 패널만). Google Docs, Arc 브라우저는 별도 설정 필요.
- **시스템 요구사항**: **macOS 14+, Apple Silicon 전용** (Intel Mac 미지원).
- **메모리 부담**: 8GB RAM 기기에서는 LLM 상주로 체감 가능. 복잡한 웹앱에서 가끔 오동작 보고.

## 4. 가격
- **현재 베타 기간 완전 무료** — 이메일 등록 후 프리릴리즈 다운로드.
- **정식 출시 계획** (잠정):

| 플랜 | 가격 | 내용 |
|---|---|---|
| Casual | 무료 | 제한적 자동완성 (경량 사용) |
| Plus | 월 구독 (미공개) | 1대 기기 무제한 |
| Pro | 월 구독 (미공개) | 최대 3대, 커스텀 지시, 개인 문체 학습 |

- Pro 30일 체험 예정.

## 5. 경쟁 포지션
전통적 텍스트 확장기(TextExpander, PhraseExpress, Typinator — 고정 단축어 → 정해진 스니펫 치환)와 결이 다름. **정적 치환이 아닌 문맥 기반 동적 생성**이 차별점. Apple의 예측 텍스트보다 공격적이고 범용적이며, 클라우드형 AI 라이터(Grammarly, Copilot)와 달리 **로컬 처리 + 앱 구애 없이 어디서나**가 강점.

## 6. 리뷰 종합 (Brave × Perplexity 교차)
- **Lifehacker / Yahoo Tech**: "모든 Mac 앱에 모바일식 자동완성을 추가" — 무료·오프라인 LLM 점을 긍정 평가.
- **MacSparky (David Sparks)**: "신뢰할 수 있는 Mac 개발자, 빠르고 프라이버시 철저". 버전 0.7.2 개선 언급, 주력 도구로 사용 중.
- **Reddit r/macapps**: 개발자 직접 AMA. 로컬 실행 + "받아쓰기보다 사고 속도에 맞춰 타이핑 가능" 철학 강조.
- **Implicator.ai**: "자동완성·작문 보조·코드 완성 사이의 애매한 시장을 선점. Apple이 내놓기 전 먼저 움직인 사례". 약점은 RAM 사용과 일부 웹앱 오동작.
- **긍정**: 타이핑 속도 최대 ~50% 단축, 비원어민 작성자에게 특히 유용, 설치 즉시 체감.
- **부정/주의**: 초반에 제안이 눈에 띄어 집중 방해, OmniFocus 등 Tab을 쓰는 앱과 충돌 (제외 설정으로 해소).

## 7. 결론 (적합성 판단)
- **적합**: Apple Silicon + macOS 14+ 사용자 중 글쓰기·이메일·노트 작성량 많은 사람, 프라이버시 민감 사용자, 비원어민.
- **부적합**: Intel Mac, RAM 8GB 환경, 주력 업무가 코드 에디터인 개발자(여기서는 Copilot 계열이 우위).
- **지금 타이밍**: 베타 무료 구간이라 **리스크 없이 평가 가능**. 향후 유료 전환 예정이므로 초기 사용자로 합류해 개인 스타일 학습을 축적해두는 것이 이득.

# Cotypist 설치 방법

## 1. 시스템 요구사항 확인
- **Mac**: Apple Silicon (M1 / M1 Pro / M2 / M3 / M4 …) — **Intel Mac 불가**
- **macOS**: 14 (Sonoma) 이상
- **권장 사양**: M1 Pro 또는 M2 이상 + RAM 16GB
- **예상 메모리 점유**: 실행 시 약 1~2GB

## 2. 다운로드
1. 브라우저로 **https://cotypist.app** 접속
2. 첫 화면의 **"Free Download"** 또는 **"Get the Download Link"** 버튼 클릭
3. 이메일 주소 입력 → 다운로드 링크가 메일로 전송됨
4. 메일에서 링크를 눌러 `.dmg` (또는 `.zip`) 파일 내려받기

> 현재 베타 기간이라 **무료**이며, Mac App Store가 아닌 개발자 웹사이트에서 직접 배포.

## 3. 설치
1. 내려받은 `.dmg` 더블 클릭 → 마운트
2. **Cotypist 아이콘**을 `Applications` 폴더로 드래그
3. `Applications`에서 Cotypist 실행
4. 최초 실행 시 "확인되지 않은 개발자" 경고가 나오면 **시스템 설정 → 개인정보 보호 및 보안 → "확인 없이 열기"** 클릭

## 4. 필수 권한 부여 (중요)
Cotypist는 모든 앱의 텍스트 입력을 감지해야 하므로 **접근성 권한**이 필수.

1. **시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용(Accessibility)**
2. 목록에 추가된 **Cotypist** 토글을 **켜짐**으로
3. 앱이 권한 요청 창을 다시 띄우면 "시스템 설정 열기" → 위 과정 반복
4. (일부 앱) **입력 모니터링(Input Monitoring)** 권한도 요청될 수 있음 — 동일하게 허용

## 5. 로컬 모델 다운로드
- 최초 실행 시 온디바이스 LLM(Qwen 2.5 / Llama 3.2 / Gemma 2 중 기본 모델)을 **자동 다운로드**
- 용량은 수 GB 수준, 네트워크 환경에 따라 몇 분 소요
- 메뉴바 아이콘에서 진행 상황 확인 가능

## 6. 앱별 추가 설정
- **Google Docs**: 설정에서 **Accessibility Mode**(접근성 모드) 활성화 필요
- **Arc / Dia 브라우저**: 브라우저 별도 토글 ON 필요
- **코드 에디터(VS Code 등)**: 본문 편집기는 미지원, 사이드바 채팅/AI 패널만 작동
- **터미널(iTerm, Terminal)**: AI 프롬프트 입력으로 인식될 때만 자동 활성화

## 7. 첫 사용 확인
1. Apple Notes나 Mail을 열어 아무 문장이나 타이핑
2. 커서 뒤에 흐릿한 회색 제안 텍스트가 뜨면 정상
3. **Tab** 키로 수락 · **Esc** 키로 무시
4. 메뉴바 Cotypist 아이콘에서 일시정지, 제외 앱 추가, 통계 확인 가능

## 8. 제거(참고)
- `Applications`에서 Cotypist를 휴지통으로 이동
- 시스템 설정 → 손쉬운 사용 목록에서 Cotypist 항목 삭제
- 모델/데이터: `~/Library/Application Support/Cotypist` 수동 삭제
