# 클로드 코드: 단순한 챗봇을 넘어선 AI 업무 엔진

클로드 코드(Claude Code)는 오늘날 가장 강력한 AI입니다. 특히 Opus 4.6 출시 이후 더욱 그렇습니다.
이것이 6개월 만에 연간 반복 매출(ARR) 10억 달러를 달성한 이유는 단 하나입니다.
이는 챗 앱(chat app)이 아니라 업무 엔진(work engine)이기 때문입니다.

대부분의 사람들은 여전히 브라우저에서 클로드(Claude)를 사용합니다. 이는 슈퍼컴퓨터를 계산기처럼 사용하는 것과 같습니다.

클로드 코드(Claude Code)가 다른 점은 다음과 같습니다.
*   완전한 시스템 접근 권한을 가집니다.
*   수 시간 동안 지속되는 작업을 실행할 수 있습니다.
*   여러 에이전트(agent)를 동시에 실행할 수 있습니다.
*   사용자의 코드베이스(codebase) 및 도구에 직접 연동됩니다.

챗 클로드(Chat Claude)는 이러한 작업을 수행할 수 없습니다. 코워크(Cowork)는 단지 얇은 레이어(layer)에 불과합니다.

다음 두 가이드부터 시작해 보세요.
*   클로드(Claude) 가이드
*   코워크(Cowork) 가이드

클로드 코드(Claude Code)가 실제로 할 수 있는 일은 놀랍습니다.
*   수 분 만에 10,000개의 파일을 스캔할 수 있습니다.
*   평이한 영어로 원하는 것을 지시할 수 있습니다.
*   사용자가 구문을 몰라도 배시(bash) 명령을 실행합니다.
*   하나의 인터페이스(interface)로 200개 이상의 도구를 연결합니다.

이는 마치 소규모 팀을 관리하는 것과 같은 느낌을 줍니다.

기본 사항부터 먼저 배우세요.
*   클로드 코드(Claude Code) 기본 가이드
*   고급 워크플로우(workflow)

핵심 워크플로우(workflow)는 다음과 같습니다.
*   분석 (Analyze)
*   계획 (Plan)
*   생성 (Create)
*   확장 (Scale)

하나의 회의록이 제품 요구 사항 문서(PRD)로 변환됩니다. PRD는 지라(Jira) 티켓이 되고, 티켓은 슬랙(Slack) 업데이트와 대시보드(dashboard)가 됩니다.
이 모든 것이 하나의 도구 안에서 이루어집니다.

*   AI PRD 가이드
*   AI 로드맵(Roadmap) 가이드

시작하는 것은 간단합니다.
1.  커서(Cursor)를 사용합니다.
2.  터미널(terminal)을 엽니다.
3.  `claude`를 입력합니다.
4.  엔터(Enter)를 누릅니다.

macOS 13 이상, Ubuntu 20 이상, Windows 10 이상에서 지원됩니다.
요금제는 월 20달러에서 200달러까지 다양합니다.

*   전문가 설정 가이드
*   공식 설정

진정한 힘은 MCP에 있습니다.
AI 도구를 위한 USB-C라고 생각하시면 됩니다.
하나의 프로토콜(protocol)로 수백 가지 통합(integration)을 제공합니다.
GitHub, Slack, Notion, Jira, 데이터베이스(database), 이메일 등이 포함됩니다.

여기서 시작해 보세요.
*   PM 운영 체제
*   앤트로픽(Anthropic) MCP 서버
*   MCP 컬렉션 (mcp.so)

필수 명령어를 미리 익히세요.
*   `/help`
*   `/clear`
*   `/compact`
*   `/model`
*   `/mcp`
파일과 폴더를 참조하려면 `@`를 사용하세요.

프롬프팅(prompting)은 사람들이 생각하는 것보다 더 중요합니다.
*   먼저 계획하세요.
*   실행 전에 검토하세요.
*   체크포인트(checkpoint)를 생성하세요.
*   문제가 발생하면 되감기(rewind)하세요.
*   명확한 피드백(feedback)으로 반복(iterate)하세요.

*   프롬프트 엔지니어링(Prompt engineering)
*   모범 사례(Best practices)

마지막으로 스킬(Skills)과 CLAUDE.md를 배우세요.
스킬(Skills)은 재사용 가능한 자동화(automation) 기능입니다.
CLAUDE.md는 장기 프로젝트 메모리(long term project memory)입니다.
이것이 진지한 팀들이 승리하는 방법입니다.

*   스킬(Skills) 튜토리얼(tutorial)
*   컨텍스트 엔지니어링(Context engineering)

클로드 코드(Claude Code)는 단지 강력한 것 이상입니다.
이는 업무 수행 방식을 변화시킵니다.
