# AI 에이전트를 위해 설계된 브라우저 자동화 CLI 출시

AI 개발자에게 브라우저 상호작용은 AI 에이전트(AI Agent)를 개발할 때 매우 빈번하게 발생하는 과정입니다. 하지만 AI 에이전트가 브라우저를 효율적으로 조작하도록 만드는 것은 상당히 까다로운 과제입니다. 최근 Vercel은 AI 에이전트 전용으로 설계된 오픈소스 브라우저 자동화 CLI(Command Line Interface)인 **agent-browser**를 공개했습니다.

### agent-browser의 주요 특징
*   **참조(Ref) 기반:** 스냅샷을 찍을 때 참조(ref) 값이 포함된 접근성 트리(accessibility tree)를 반환하여 요소에 쉽게 접근할 수 있습니다.
*   **AI 에이전트 최적화:** AI 컨텍스트를 효율적으로 활용하여 토큰(token) 소비를 줄여줍니다.
*   **세션 관리:** 별도의 인증 정보가 포함된 여러 개의 독립된 브라우저 인스턴스를 지원합니다.
*   **강력한 기능:** 페이지 탐색, 폼(form) 조작, 스크린샷 등 50개 이상의 명령어를 지원합니다.
*   **크로스 플랫폼:** macOS, Linux, Windows 환경을 모두 지원합니다.

### 설치 방법

`agent-browser`를 설치하기 전에 시스템에 Node.js가 설치되어 있어야 합니다. 터미널에서 아래 명령어를 입력하여 전역(global)으로 설치합니다.

```bash
npm install -g agent-browser
```

설치가 완료되면 아래 명령어를 실행하여 Chromium 브라우저를 다운로드합니다.

```bash
agent-browser install
```

### 주요 사용법

**1. 웹페이지 열기**
```bash
agent-browser open https://agent-browser.dev/
```

**2. 현재 웹페이지의 접근성 트리(accessibility tree) 가져오기**
```bash
agent-browser snapshot -i
```
*출력 예시:*
- link "agent-browser" [ref=e2]
- button "Ask AI" [ref=e26]

**3. 참조(ref) 값을 사용한 상호작용**
```bash
agent-browser click @e3
```

**4. 스크린샷 캡처**
```bash
agent-browser screenshot ./page.png
```

**5. 브라우저 종료**
```bash
agent-browser close
```

### 스킬(Skills) 설치 및 AI 연동

`agent-browser`는 AI 에이전트를 위해 설계되었으므로, 이를 활용하면 브라우저 작업을 손쉽게 자동화할 수 있습니다. Cursor, Claude Code, Codex 등에서 브라우저를 직접 제어하려면 아래 명령어로 스킬을 추가합니다.

```bash
npx skills add vercel-labs/agent-browser --skill agent-browser
```

예를 들어, Claude Code를 사용하여 특정 웹사이트의 스크린샷을 찍도록 명령하면, AI가 자동으로 `agent-browser`를 실행하여 다음과 같이 작업을 수행합니다.

```bash
❯ Save a screenshot of the https://agent-browser.dev/ webpage to the current directory.

⏺ Skill(agent-browser) 로딩 완료
⏺ Bash 실행 (open -> wait -> screenshot)
✓ 브라우저 실행 및 스크린샷 저장 완료
```

### 요약

`agent-browser`는 단순히 CLI 기능뿐만 아니라 iOS 시뮬레이터의 Safari 브라우저를 제어하여 모바일 웹 테스트를 수행하거나, CDP(Chrome DevTools Protocol) 모드를 통해 로컬 또는 원격 브라우저에 연결하는 등 확장성이 뛰어납니다. AI 에이전트가 웹을 더 잘 이해하고 조작할 수 있는 환경을 구축하고자 한다면 매우 유용한 도구가 될 것입니다.
