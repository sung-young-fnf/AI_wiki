## 1. Aider — 업계 표준 AI 페어 프로그래머

Aider는 아마도 이 목록에서 가장 강력한 도구일 것입니다. 터미널에서 직접 대규모 언어 모델(LLM)(예: GPT-4o, Claude 3.5 또는 Ollama를 통한 로컬(local) 모델)과 페어 프로그래밍(pair-program)을 할 수 있도록 해줍니다.

단순한 채팅과 달리, Aider는 파일을 실시간으로 편집하고 변경 사항마다 설명이 포함된 Git 커밋(commit)을 자동으로 생성합니다. 전체 리포지토리(repository)의 컨텍스트(context)를 매핑(map)하여, 한 파일의 변경이 다른 파일에 어떻게 영향을 미치는지 정확히 파악합니다.

**놀라운 점**: 말 그대로 “로직(logic)을 리팩토링(refactor)하여 JWT를 사용하도록 해줘”라고 말하면, 커피를 다 마시기도 전에 5개 파일을 편집하고 변경 사항을 커밋하는 것을 볼 수 있습니다.

또한 “음성-코드 변환(Voice-to-Code)” 기능을 지원하며, 이를 통해 Aider에게 코드에 대해 말하고 개선 사항이나 버그 수정(bug fix)을 요청할 수 있습니다.

Aider를 시작하는 방법은 다음과 같습니다.

```bash
python -m pip install aider-install
aider-install

# Change directory into your codebase
cd /to/your/project

# DeepSeek
aider --model deepseek --api-key deepseek=<key>

# Claude 3.7 Sonnet
aider --model sonnet --api-key anthropic=<key>

# o3-mini
aider --model o3-mini --api-key openai=<key>
```

## 2. Gemini CLI — Google의 에이전트형 셸

구독 없이도 고성능 AI 파워를 원한다면, Gemini CLI는 현재 무료 티어(free tier)의 왕입니다. Gemini 3 Flash 및 Pro에 터미널에서 직접 접근할 수 있도록 해줍니다.

이는 “에이전트형(Agentic)” 작업을 위해 구축되었습니다. 명령을 실행하고, 파일 시스템(file system)을 탐색하며, 실시간으로 오류를 수정할 수 있습니다.

CLI 내부에 멋진 인터페이스(interface)를 갖추고 있어, Gemini CLI는 2026년에 꼭 사용해봐야 할 도구 중 하나입니다!

게다가, 비디오 생성(Video Generation), 심층 연구(Deep Research), 노트북 LM(Notebook LM) 등 Google에서 제공하는 모든 다양한 AI 도구에 대한 접근을 제공합니다!

Gemini CLI는 npm을 사용하여 설치할 수 있습니다.

```bash
npm install -g @google/gemini-cli
```

더 많은 설치 방법은 해당 웹사이트를 참조하세요!

## 3. Atuin — ‘두뇌’를 갖춘 셸 기록

3주 전에 실행했던 명령을 기억하는 것은 `history | grep` 게임과 같았습니다. Atuin은 셸 기록(shell history)을 검색 가능한 SQLite 데이터베이스(database)로 대체하며, 이는 여러 장치 간에 동기화(sync)됩니다.

AI 시대에 Atuin은 진화했습니다. 이제 의미론적 검색(semantic search)과 AI 기반 명령 발견을 지원합니다. 정확한 플래그(flag)를 기억하는 대신, 자신이 무엇을 하려 했는지의 ‘의도(intent)’를 검색할 수 있습니다.

다음 `curl` 명령을 사용하여 Atuin을 설치하세요.

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://setup.atuin.sh | sh
```

더 자세한 정보는 해당 웹사이트를 방문하세요.

## 4. ShellGPT (sgpt) — AI 셸의 맥가이버 칼

때로는 Aider와 같은 완전한 페어 프로그래머가 필요하지 않을 때도 있습니다. 단지 빠른 답변이 필요할 뿐이죠. ShellGPT는 ChatGPT(또는 호환 가능한 API)를 파이프라인(pipeline)에 통합하는 경량 도구입니다.

로그(log)를 직접 파이프하여 전달할 수 있습니다: `cat error.log | sgpt "explain this error"`, 또는 즉석에서 셸 스크립트(shell script)를 생성하도록 요청할 수 있습니다. 심지어 스테이징(staging)된 변경 사항을 기반으로 Git 커밋 메시지를 생성할 수도 있습니다.

shell-gpt를 시작하려면 pip를 사용하세요.

```bash
pip install shell-gpt

# Generate a command and run it
sgpt --shell "find all large files in /var/log"
```

## 5. Warp — 에이전트형 개발 환경

Warp는 다수의 AI 에이전트와 함께 코딩할 수 있도록 구축된 에이전트형 개발 환경입니다. Warp는 현대적인 사용자 인터페이스(UI)와 코드 편집 기능을 통해 터미널을 21세기로 가져옵니다. Warp의 최첨단(SOTA) 내장 에이전트 Oz를 사용하거나, Claude Code, Codex 또는 Gemini CLI와 같은 CLI 코딩 에이전트를 실행할 수 있습니다.

**무료 티어(Free Tier)**: Warp는 개인 사용자에게는 무료이며, 매월 상당량의 AI 요청을 포함합니다.

## 6. Goose — 오픈소스 ‘바이브 코딩’ 에이전트

Goose는 복잡한 개발 작업을 처음부터 끝까지 자동화할 수 있는 온머신(on-machine) AI 에이전트입니다. 단순한 코드 제안을 넘어, Goose는 전체 프로젝트를 처음부터 구축하고, 코드를 작성 및 실행하며, 오류를 디버그(debug)하고, 워크플로우를 조정하며, 외부 API(Application Programming Interface)와 자율적으로 상호 작용할 수 있습니다.

최대한의 유연성을 위해 설계된 Goose는 어떤 LLM과도 작동하며, 성능과 비용을 최적화하기 위한 다중 모델 구성(multi-model configuration)을 지원합니다. MCP 서버(server)와 완벽하게 통합되며, 데스크톱 앱(desktop app)과 CLI 모두로 제공됩니다. 이는 더 빠르게 움직이고 혁신에 집중하고자 하는 개발자를 위한 궁극적인 AI 어시스턴트(assistant)입니다.

Mac에 Goose를 설치하려면 다음 brew 명령을 사용하세요.

```bash
brew install block-goose-cli
```

더 많은 설치 방법은 해당 웹사이트를 참조하세요.

## 7. OpenCode — 프라이버시 우선 대안

코드를 클라우드 제공업체(cloud provider)로 보낼 수 없는 민감한 프로젝트(project)를 작업하는 경우, OpenCode는 최고의 친구가 될 것입니다. 이는 완전히 로컬(local) 모델을 포함하여 75개 이상의 다양한 LLM 제공업체(provider) 간에 쉽게 전환할 수 있도록 하는 커뮤니티(community) 기반 도구입니다.

이는 공급업체 종속(vendor lock-in) 없이 ‘에이전트’ 경험을 원하는 개발자를 위해 설계되었습니다.

다음은 OpenCode를 시작하는 몇 가지 방법입니다.

```bash
# YOLO
curl -fsSL https://opencode.ai/install | bash

# Package managers
npm i -g opencode-ai@latest        # or bun/pnpm/yarn
scoop install opencode             # Windows
choco install opencode             # Windows
brew install anomalyco/tap/opencode # macOS and Linux (recommended, always up to date)
brew install opencode              # macOS and Linux (official brew formula, updated less)
sudo pacman -S opencode            # Arch Linux (Stable)
paru -S opencode-bin               # Arch Linux (Latest from AUR)
mise use -g opencode               # Any OS
nix run nixpkgs#opencode           # or github:anomalyco/opencode for latest dev branch
```
