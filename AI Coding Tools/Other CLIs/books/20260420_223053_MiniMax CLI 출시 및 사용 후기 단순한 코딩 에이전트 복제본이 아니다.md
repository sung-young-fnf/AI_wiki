# MiniMax CLI 출시 및 사용 후기: 단순한 코딩 에이전트 복제본이 아니다

다른 Claude Code 대체 CLI(Command Line Interface)와 달리 MiniMax CLI는 독특한 점을 제공합니다.

하지만 처음에는 경쟁에 발맞추기 위한 급조된 도구이거나 Claude Code, Aider, Codex CLI의 유행에 편승하려는 또 다른 시도라고 착각했습니다.

저는 틀렸습니다.

MiniMax CLI는 코딩 에이전트가 아닙니다. 코드를 작성하거나 파일을 리팩터(refactor)하거나 레포지토리(repository)에 변경 사항을 커밋(commit)하지 않습니다. 이는 터미널(terminal)에서 텍스트, 이미지, 비디오, 음성 및 음악을 생성할 수 있는 멀티모달(multimodal) 생성 도구입니다.

참고로 MiniMax는 2026년 1월 홍콩 증권 거래소에 상장된 상하이 기반의 AI 회사입니다.

이들은 다음과 같은 제품을 구동하는 멀티모달 기반 모델(foundation models)을 구축했습니다:
*   Hailuo AI (비디오 생성)
*   Talkie (AI 캐릭터 채팅)

최근 MiniMax의 M2.7 모델이 코딩 벤치마크(benchmark)에 등장하기 시작했고, 이는 제가 이 CLI를 또 다른 코딩 도구라고 생각했던 이유를 설명해 줄 것입니다. 저는 최근 Claude Code에서 MiniMax 2.7을 테스트하며 Claude Code 토큰을 절약하는 방법을 찾았습니다.

저는 MiniMax CLI의 각 기능, 즉 텍스트 생성, 이미지 생성, 음성 합성(speech synthesis), 음악 생성, 비디오 출력, 비전 분석(vision analysis) 및 웹 검색을 테스트하는 데 시간을 보냈습니다. 이 글은 어떤 기능이 작동하는지, 그리고 이 도구가 여러분의 작업 흐름(workflow)에 포함될 만한 가치가 있는지 분석합니다.

## MiniMax CLI란?

MiniMax CLI는 다음 7가지 기능을 중심으로 구축된 터미널 도구입니다.

*   **텍스트 (Text)** — 다중 턴 채팅(multi-turn chat), 스트리밍 응답(streaming responses), 시스템 프롬프트(system prompts), JSON(제이슨) 출력
*   **이미지 (Image)** — 가로세로 비율(aspect ratio) 및 배치 제어(batch controls)를 통한 텍스트-이미지 변환(text-to-image) 생성
*   **비디오 (Video)** — 진행 상황 추적 기능을 갖춘 비동기(async) 비디오 생성
*   **음성 (Speech)** — 30개 이상의 음성 및 속도 제어 기능을 갖춘 텍스트-음성 변환(text-to-speech)
*   **음악 (Music)** — 선택적 가사 또는 연주곡 모드(instrumental mode)를 통한 텍스트-음악 변환(text-to-music)
*   **비전 (Vision)** — 이미지 이해 및 설명
*   **검색 (Search)** — MiniMax 기반의 웹 검색

이 모든 기능은 단일 `mmx` 명령을 통해 실행됩니다. 한 번 인증하면 필요한 도구를 호출할 수 있습니다.

MiniMax는 이 CLI를 "에이전트 지향적(agent-oriented)"으로 구축했습니다. 이는 AI 에이전트(agent)의 사용에 최적화되어 있습니다. 작업 중에 이미지를 생성하거나 음성을 합성해야 하는 자율 에이전트(autonomous agent)를 구축하고 있다면 이 CLI가 유용할 것입니다.

터미널에서 전역적으로 설치할 수 있습니다:

```bash
npm install -g mmx-cli
```

또는 Cursor, Claude Code, OpenClaw와 같은 AI 에이전트에 스킬(skill)로 추가할 수 있습니다:

```bash
npx skills add MiniMax-AI/cli -y -g
```

두 번째 옵션을 사용하면 코딩 에이전트가 미디어를 생성하고, 이미지를 분석하며, 웹을 검색하는 기능을 얻게 됩니다.

## 설치 및 설정

Node.js 18 이상이 필요합니다. 이를 갖추고 있다면 설치는 하나의 명령으로 완료됩니다:

```bash
npm install -g mmx-cli
```

다음으로 MiniMax API 키(API key)가 필요합니다. MiniMax 플랫폼으로 이동하여 계정을 만드세요. API는 무료가 아니므로 토큰 플랜(Token Plan) 구독도 필요합니다.

키를 얻었다면 인증합니다:

```bash
mmx auth login --api-key sk-xxxxx
```

또는 키 플래그(key flag) 없이 `mmx auth login`을 실행하면 브라우저 OAuth 플로우(OAuth flow)가 열립니다.

모든 것이 제대로 작동하는지 확인하려면:

```bash
mmx auth status
```

남은 할당량(quota)도 언제든지 확인할 수 있습니다:

```bash
mmx quota
```

참고할 점은 MiniMax는 두 가지 지역에서 운영됩니다. 글로벌 사용자는 `api.minimax.io`를 사용하고 중국 사용자는 `api.minimaxi.com`을 사용합니다. CLI는 이를 자동으로 처리하지만, 필요한 경우 수동으로 설정할 수 있습니다:

```bash
mmx config set --key region --value global
```

## 기능 테스트

각 명령과 그로 할 수 있는 작업에 대한 분석입니다.

### 텍스트 생성 (Text Generation)

기본 채팅은 다른 LLM(대규모 언어 모델) CLI와 유사하게 작동합니다:

```bash
mmx text chat --message "Explain microservices in simple terms"
```

동작을 제어하기 위해 시스템 프롬프트(system prompt)를 추가할 수 있습니다:

```bash
mmx text chat --system "You are a senior backend engineer" --message "Review this architecture"
```

스트리밍 응답(streaming responses)의 경우:

```bash
mmx text chat --model MiniMax-M2.7-highspeed --message "Write a product description" --stream
```

다중 턴 대화(multi-turn conversations)를 위해 파일에서 메시지를 파이프(pipe)할 수도 있습니다:

```bash
cat messages.json | mmx text chat --messages-file - --output json
```

### 이미지 생성 (Image Generation)

간단한 프롬프트(prompt)로 이미지를 생성합니다:

```bash
mmx image "A futuristic city at sunset"
```

가로세로 비율(aspect ratio) 및 배치 크기를 제어합니다:

```bash
mmx image generate --prompt "Logo design for a tech startup" --n 3 --aspect-ratio 16:9
```

특정 디렉터리에 저장합니다:

```bash
mmx image generate --prompt "Abstract art" --out-dir ./images/
```

### 비디오 생성 (Video Generation)

비디오 생성은 렌더링(rendering)에 시간이 걸리므로 비동기(asynchronously)로 실행됩니다:

```bash
mmx video generate --prompt "Ocean waves crashing on rocks" --async
```

작업 상태를 확인합니다:

```bash
mmx video task get --task-id 123456
```

준비되면 다운로드합니다:

```bash
mmx video download --file-id 176844028768320 --out video.mp4
```

### 음성 합성 (Speech Synthesis)

텍스트를 음성으로 변환합니다:

```bash
mmx speech synthesize --text "Welcome to the future of AI" --out welcome.mp3
```

30개 이상의 음성 중에서 선택하고 속도를 조절합니다:

```bash
mmx speech synthesize --text "Breaking news update" --voice English_magnetic_voiced_man --speed 1.2
```

오디오를 직접 스트리밍(stream)합니다:

```bash
mmx speech synthesize --text "Stream this" --stream | mpv -
```

### 음악 생성 (Music Generation)

가사를 포함한 음악을 생성합니다:

```bash
mmx music generate --prompt "Upbeat pop song" --lyrics "[verse] Sunny days ahead" --out song.mp3
```

또는 연주곡 트랙(instrumental tracks)을 만듭니다:

```bash
mmx music generate --prompt "Cinematic orchestral score" --instrumental --out background.mp3
```

### 비전 (이미지 분석) (Vision (Image Analysis))

로컬 이미지를 분석합니다:

```bash
mmx vision photo.jpg
```

또는 특정 질문과 함께 URL(Uniform Resource Locator)을 전달합니다:

```bash
mmx vision describe --image https://example.com/image.jpg --prompt "What objects are in this image?"
```

### 웹 검색 (Web Search)

검색 쿼리(query)를 실행합니다:

```bash
mmx search "MiniMax AI latest updates"
```

JSON 형식으로 결과를 얻습니다:

```bash
mmx search query --q "latest AI news" --output json
```

## 최종 의견

MiniMax CLI는 이미지, 오디오, 비디오와 같은 미디어를 생성해야 하는 AI 에이전트를 구축하는 개발자를 위한 공백을 메워줍니다.

저는 Claude Code, Gemini CLI 또는 Codex처럼 작동하는 또 다른 CLI를 예상했지만, MiniMax는 코딩 CLI가 아닙니다.

이러한 접근 방식은 CLI 도구가 코딩 외에 훨씬 더 많은 작업을 수행하도록 발전할 수 있는 미래를 암시하기도 합니다. MiniMax 2.7은 코딩에 훌륭한 모델입니다.
