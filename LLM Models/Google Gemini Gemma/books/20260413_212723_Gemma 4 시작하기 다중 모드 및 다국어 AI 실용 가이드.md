## Gemma 4 시작하기: 다중 모드 및 다국어 AI 실용 가이드

제로부터 추론(inference)까지: Kaggle에서 Google의 Gemma 4 2B 모델을 이미지, 비디오, 언어 전반의 실제 사례와 함께 탐색합니다.

Gemma 4가 출시되었습니다.

### 서론

Gemma 4의 출시는 개방형(open), 효율적이고 유능한 AI 모델의 발전에 있어 또 다른 중요한 진전을 의미합니다. Google이 개발한 Gemma 모델은 가볍지만 강력하게 설계되어, 대규모 인프라 없이도 뛰어난 성능을 원하는 개발자에게 이상적입니다.

Gemma 4는 다음 네 가지 모델 크기를 선보입니다:
*   E2B, E4B — 최대 컴퓨팅 및 메모리 효율성을 위한 모델
*   26B (MoE), 31B — 매개변수당(per parameter) 최고 수준의 지능을 제공하는 모델

Gemma 4는 고급 추론(reasoning) 및 에이전트(agentic) 워크플로를 위해 특별히 구축된 개방형 모델 제품군입니다.

이 실습 가이드에서는 Kaggle 노트북을 사용하여 Gemma 4 2B 다중 모드(multimodal), 다국어(multilingual) 모델을 살펴볼 것입니다. 목표는 간단합니다. 최대한 빠르게 모델 설정부터 실제 사용 사례까지 안내하여 모델을 평가할 수 있도록 돕는 것입니다.

### 단계별 안내

Kaggle 노트북에서 Gemma를 실행하려면, Gemma 4가 최근에 출시되었으므로(몇 시간 전 출시), `transformers` 라이브러리를 최신 버전으로 업그레이드해야 합니다.

```python
!pip install -U transformers
```

이를 통해 `transformers` 5.5.0 버전으로 환경이 업그레이드됩니다 (이 글 작성 시점 기준).

다음으로, 라이브러리를 포함하고 `transformers` 파이프라인(pipeline)을 초기화합니다.

```python
from transformers import pipeline
pipe = pipeline("any-to-any", model="google/gemma-4-e2b-it") 
```

다중 모드 입력을 허용하기 위해 `any-to-any` 옵션으로 파이프라인을 초기화합니다.

모델 다운로드가 완료되면 테스트를 시작할 수 있습니다. 여기서는 모든 테스트를 재현하지는 않을 것입니다. Kaggle 노트북의 예시 중에서 이미지 테스트 하나와 비디오 테스트 하나를 선택하여 진행하겠습니다 (Gemma 4 기술 보고서에 있는 내용).

### 독일어로 이미지 테스트

우리가 해석할 이미지는 여러 색깔의 사탕입니다.

모델에 독일어로 사탕의 색깔이 무엇인지 질문합니다.

```python
messages = [
    {
        "role": "user",
        "content": [
            {"type": "image", "url": "https://huggingface.co/datasets/huggingface/documentation-images/resolve/main/p-blog/candy.JPG"},
            {"type": "text", "text": "Welche Farben haben die Bombons?"}
        ]
    }
]
output = pipe(text=messages, max_new_tokens=200)
```

```python
display(Markdown(output[0]["generated_text"][-1]["content"]))
```

결과는 다음과 같습니다:

(독일어 원문)
```
Die Bombons auf dem Bild haben folgende Farben:

Türkisgrün/Blaugrün (zwei Stück)

Orange (ein Stück)

Grün (ein Stück)
```
(번역: 이미지의 사탕들은 다음 색상을 가집니다: 터키석/청록색 (두 개), 주황색 (한 개), 녹색 (한 개))

결과는 양호하며, 사탕의 색깔과 특정 색깔 사탕의 개수가 정확하게 식별되었습니다.

### 비디오 테스트

다음으로, 발사 전 로켓 비디오를 확인해 보겠습니다.

```python
messages = [
    {
        "role": "user",
        "content": [
            {
                "type": "video",
                "video": "https://huggingface.co/datasets/merve/vlm_test_images/resolve/main/rockets.mp4",
            },
            {"type": "text", "text": "What is happening in this video?"},
        ],
    }
]

output = pipe(messages, load_audio_from_video=True)
```

```python
display(Markdown(output[0]["generated_text"][-1]["content"]))
```

모델의 답변은 다음과 같습니다:

(영어 원문)
```
This video shows a crowd of people gathered on a tarmac, observing a large rocket, which appears to be a SpaceX Falcon 9 rocket, likely before or after a launch event. The sky is cloudy with soft, diffused light, suggesting either sunrise or sunset. There are also other aircraft and what look like exhibition booths in the background.
```
(번역: 이 비디오는 활주로에 모여든 사람들이 큰 로켓을 관찰하는 모습을 보여줍니다. 이 로켓은 스페이스X(SpaceX) 팔콘 9 로켓으로 보이며, 발사 전후 상황으로 추정됩니다. 하늘은 구름으로 덮여 부드럽고 확산된 빛을 띠어 일출이나 일몰을 암시합니다. 배경에는 다른 항공기와 전시 부스로 보이는 것들도 있습니다.)

답변은 비디오의 세부 사항을 정확하게 식별합니다. 로켓의 제조사와 종류에 대한 작은 오류가 있지만, 두 로켓은 매우 유사합니다.

### 일본어로 일본 랜드마크 테스트

이제 일본의 랜드마크를 사용하여 모델을 테스트해 보겠습니다. 질문도 일본어로 할 것입니다. 이미지는 도쿄 타워입니다.

```python
messages = [
    {
        "role": "user",
        "content": [
            {"type": "image", "url": "https://www.advantour.com/img/japan/tokyo/tokyo-tower.jpg"},
            {"type": "text", "text": "この画像には何が見えますか?"}
        ]
    }
]
output = pipe(text=messages, max_new_tokens=200)
```

```python
display(Markdown(output[0]["generated_text"][-1]["content"]))
```

모델이 출력한 답변은 다음과 같습니다:

(일본어 원문)
```
この画像は、晴れた日の都市のパノラマビューを捉えています。

主な特徴:

東京のスカイライン: 多くの高層ビルが立ち並ぶ都市の景色が広がっています。

東京タワー: 画像の中央やや左寄りに、象徴的な赤い東京タワーがそびえ立っています。

空: 非常に青く澄んだ快晴の空が広がり、明るい日差しの下で撮影されたことがわかります。

植生: 都市の中にも緑が多く、特に前景や中景には木々（秋の紅葉を思わせる色合いも見られます）が見られ、自然と都市が共存している様子がうかがえます。

建築物: 様々な高さとデザインの近代的なビル群が密集しています。
```
(번역)
이 이미지는 맑은 날 도시의 파노라마 전경을 담고 있습니다.

주요 특징:

*   도쿄 스카이라인: 많은 고층 빌딩이 늘어선 도시 풍경이 펼쳐집니다.
*   도쿄 타워: 이미지 중앙에서 약간 왼쪽에 상징적인 붉은 도쿄 타워가 솟아 있습니다.
*   하늘: 매우 푸르고 맑은 쾌청한 하늘이 펼쳐져 있어, 밝은 햇살 아래 촬영되었음을 알 수 있습니다.
*   식생: 도시 안에도 녹지가 많으며, 특히 전경과 중경에는 나무들(가을 단풍을 연상시키는 색상도 보입니다)이 보여 자연과 도시가 공존하는 모습을 엿볼 수 있습니다.
*   건축물: 다양한 높이와 디자인의 현대적인 빌딩들이 밀집해 있습니다.

답변은 정확하며, 많은 의미 있는 세부 사항을 다루고 있어 상당히 정밀합니다.

Kaggle 노트북에서는 프랑스어, 독일어, 스페인어, 일본어로 이미지 해석 및 언어 이해 모델을 테스트하는 더 많은 예시를 찾을 수 있습니다.

### 최종 의견

"단지" 2B 매개변수(parameter)임에도 불구하고, Gemma 4는 놀라운 성능을 보여줍니다:

*   Kaggle 환경에서 빠른 추론(inference)
*   대규모 모델에 비해 낮은 메모리 점유율(memory footprint)
*   놀라운 다중 모드(multimodal) 및 다국어(multilingual) 능력

이러한 균형은 AI 기능 프로토타이핑, 데모 실행, 그리고 모바일 플랫폼에서도 잘 작동하는 경량 애플리케이션(lightweight application) 구축에 이상적입니다.
