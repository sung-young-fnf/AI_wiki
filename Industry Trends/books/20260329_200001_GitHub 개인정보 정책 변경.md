🚨 GitHub, 개인정보 정책 변경

4월 24일부터 GitHub의 AI Copilot 학습에 사용자 데이터가 활용될 수 있습니다.
비공개 데이터 학습을 원치 않으신다면, 지금 바로 비활성화해야 합니다.

---

지난 25일, GitHub은 GitHub Copilot의 상호작용 데이터(interaction data)를 자사 및 Microsoft의 AI 모델 학습에 사용할 예정임을 밝혔습니다. 수집되는 데이터에는 Copilot을 통한 입력(input), 출력(output), 코드(code), 기타 컨텍스트(context) 등이 포함됩니다. 이 정책은 개인 요금제(personal plan)에만 해당되며, 교육/비즈니스(Business)/엔터프라이즈(Enterprise) 요금제 사용자에게는 해당되지 않습니다.

기존 Copilot 모델은 공개 코드(public code)만을 활용해 만들어졌지만, 더 고도화된 모델을 구축하기 위해 실제 사용자의 사용 패턴(usage patterns), 입력 데이터(input data) 등을 추가 학습할 필요성을 느꼈다고 합니다.

Copilot에 입력된 프롬프트(prompt)와 Copilot이 작업 과정에서 읽은 모든 코드가 학습 대상에 포함됩니다. 특히 VS Code에 연동하여 탭 완성(tab completion)을 이용 중인 분들은 주의하셔야겠습니다. API 키(API key), 토큰(token), 비밀번호와 같은 민감 정보는 미리 제거하도록 다중 안전 장치를 준비했다고 합니다.

---

커뮤니티의 반응은 부정적입니다.

"수집되기 싫으면 빨리 비활성화하라고 하는 것은 다크 패턴(dark pattern)이다. 수집에 동의하면 활성화해 달라고 공지하는 것이 맞지 않나?"와 같은 의견이 지배적입니다.

대부분의 개발자가 코드를 보관하고 있는 GitHub인 만큼, 개인 정보 침해(privacy violation) 우려에 대한 비판도 거센 상황입니다.

---

비활성화는 GitHub 홈페이지의 설정(Settings) 메뉴에서 할 수 있습니다.

1.  다음 링크로 접속 (https://lnkd.in/g6_CDEGR)
2.  AI 모델 학습(AI model training)을 '비활성화(Disabled)'로 설정

학습은 오는 4월 24일부터 시작된다고 하니,
원치 않는 사용자께서는 미리 비활성화해 주시길 바랍니다!
