# 파이썬 프로젝트 성능을 극대화할 14가지 숨겨진 보석 라이브러리

프로젝트를 위해 파이썬 라이브러리를 탐색하던 중, 크게 주목받지 못했지만 매우 유용하고 과소평가된 숨겨진 보석 같은 라이브러리들을 발견했습니다. 이 가이드에서는 여러분이 반드시 알아야 할 14가지 과소평가된 파이썬 라이브러리를 소개합니다.

### 1. PyFilesystem – 파일 시스템 추상화

`fs` 라이브러리는 동일한 API를 사용하여 로컬, 클라우드 및 인메모리 파일 시스템을 다룰 수 있는 가상 파일 시스템 추상화 계층을 제공합니다.

Docs : https://www.pyfilesystem.org/

**설치**
```bash
pip install fs
```

**예시**
```python
from fs.memoryfs import MemoryFS

fs = MemoryFS()
fs.writetext('hello.txt', 'Hello, PyFilesystem!')
print(fs.readtext('hello.txt'))  # Output: Hello, PyFilesystem!
```

### 2. PyShorteners — URL 단축

`Pyshorteners`는 URL을 빠르고 쉽게 단축하는 데 도움이 되는 놀라운 숨겨진 보석 라이브러리입니다. TinyURL, Bitly 등 다양한 서비스를 지원합니다.

Docs : https://pyshorteners.readthedocs.io/en/latest/

**설치**
```bash
pip install pyshorteners
```

**예시**

TinyURL을 사용하여 URL을 단축하는 간단한 예시입니다:

```python
import pyshorteners

s = pyshorteners.Shortener()
print(s.tinyurl.short("https://www.python.org"))
```

Bitly를 사용하여 URL을 단축하는 간단한 예시입니다:

```python
import pyshorteners
s = pyshorteners.Shortener(api_key='YOUR_KEY')
s.bitly.short('http://www.google.com') # 'http://bit.ly/TEST'
s.bitly.expand('https://bit.ly/TEST') # 'http://www.google.com'
s.bitly.total_clicks('https://bit.ly/TEST') # 10
```

### 3. DeepDiff — 객체 비교

`DeepDiff`는 파이썬 객체를 비교하고 깊이 있는 수준에서 차이점을 보여줍니다.

Docs : https://pypi.org/project/deepdiff/

**설치**
```bash
pip install deepdiff
```

**예시**
```python
from deepdiff import DeepDiff

d1 = {"name": "John", "age": 25}
d2 = {"name": "John", "age": 30}

print(DeepDiff(d1, d2))  

# Output: {'values_changed': {'root['age']': {'old_value': 25, 'new_value': 30}}}
```

### 4. Yaspin — 터미널 스피너

`Yaspin`은 터미널 출력에 세련된 로딩 스피너(loading spinner)를 추가합니다.

Docs : https://github.com/pavdmyt/yaspin

**설치**
```bash
pip install yaspin
```

**예시**

터미널 출력에 로딩 스피너를 추가하는 기본적인 예시입니다:

```python
import time
from random import randint
from yaspin import yaspin

with yaspin(text="Loading", color="yellow") as spinner:
    time.sleep(2)  # time consuming code

    success = randint(0, 1)
    if success:
        spinner.ok("✅ ")
    else:
        spinner.fail("💥 ")
```

### 5. PyPDF2 — PDF 편집

`PyPDF2`는 모든 종류의 PDF 작업에 유용한 놀라운 라이브러리입니다. 텍스트 추출, PDF 병합, 워터마크 추가, PDF 분할 및 PDF 수정과 같은 기능을 제공합니다.

Docs : https://pypdf2.readthedocs.io/en/3.x/index.html

**설치**
```bash
pip install pypdf2
```

**예시**

PDF에서 텍스트를 추출하는 기본적인 예시입니다:

```python
import PyPDF2

with open("example.pdf", "rb") as f:
    reader = PyPDF2.PdfReader(f)
    print(reader.pages[0].extract_text())  # Read first page
```

### 7. TextBlob — 쉬운 자연어 처리(NLP)

`TextBlob`은 감성 분석(sentiment analysis) 및 텍스트 번역과 같은 간단한 자연어 처리(NLP) 기능을 제공합니다.

Docs : https://textblob.readthedocs.io/en/dev/

**설치**
```bash
pip install textblob
```

**예시**
```python
from textblob import TextBlob

text = """
The titular threat of The Blob has always struck me as the ultimate movie
monster: an insatiably hungry, amoeba-like mass able to penetrate
virtually any safeguard, capable of--as a doomed doctor chillingly
describes it--"assimilating flesh on contact.
Snide comparisons to gelatin be damned, it's a concept with the most
devastating of potential consequences, not unlike the grey goo scenario
proposed by technological theorists fearful of
artificial intelligence run rampant.
"""

blob = TextBlob(text)
blob.tags  # [('The', 'DT'), ('titular', 'JJ'), ('threat', 'NN'), ('of', 'IN'), ...]

blob.noun_phrases  # WordList(['titular threat', 'blob', 'ultimate movie monster', 'amoeba-like mass', ...])

for sentence in blob.sentences:
    print(sentence.sentiment.polarity) 
    
# 0.060
# -0.341
```

### 12. Lark — 초고속 파서(Parser)

`Lark`는 프로그래밍 언어와 같은 구조화된 텍스트를 파싱(parsing)하고 처리하는 파이썬 라이브러리입니다.

Docs : https://github.com/lark-parser/lark

**설치**
```bash
pip install lark
```

**예시**

"Hello, World!"를 파싱하는 간단한 프로그램입니다:

```python
from lark import Lark

l = Lark('''start: WORD "," WORD "!"

            %import common.WORD   // imports from terminal library
            %ignore " "           // Disregard spaces in text
         ''')

print( l.parse("Hello, World!") )

# output : Tree(start, [Token(WORD, 'Hello'), Token(WORD, 'World')])
```

### 13. Scapy — 네트워크 패킷 조작

`Scapy`는 네트워크 패킷을 캡처, 조작 및 전송하는 파이썬 라이브러리입니다.

Docs : https://scapy.readthedocs.io/en/latest/introduction.html

**설치**
```bash
pip install scapy
```

**예시**

패킷을 구성하고 다루는 간단한 예시입니다:

```python
from scapy.all import IP

a=IP(ttl=10)
print(a) # output : < IP ttl=10 |>

print(a.src) # output : ’127.0.0.1’

a.dst="192.168.1.1"
print(a) # output : < IP ttl=10 dst=192.168.1.1 |>

print(a) # output : ’192.168.8.14’

del(a.ttl)
print(a) # output : < IP dst=192.168.1.1 |>

print(a.ttl) # output : 64
```

Docs : https://pypi.org/project/soundfile/

**설치**
```bash
pip install soundfile
```

**예시**

오디오 파일을 읽고 쓰는 간단한 예시입니다:

```python
import soundfile as sf

data, samplerate = sf.read('existing_file.wav') # Read Audio Files
sf.write('new_file.flac', data, samplerate)     # Write Auido Files
```
