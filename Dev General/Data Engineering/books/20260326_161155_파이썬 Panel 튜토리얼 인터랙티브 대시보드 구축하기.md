## Panel이란 무엇인가요?

`Panel`은 파이썬을 사용하여 인터랙티브 웹 애플리케이션과 대시보드를 만들 수 있는 오픈 소스 파이썬 라이브러리입니다. `Jupyter Notebook` (주피터 노트북)과 원활하게 작동하며 `Matplotlib`, `Bokeh`, `Plotly`와 같은 라이브러리의 시각화를 지원합니다. `Panel`을 사용하면 데이터 시각화에 인터랙티브 위젯(예: 슬라이더, 버튼, 드롭다운)을 추가할 수 있습니다.

### 전제 조건
*   기본적인 파이썬 지식
*   `Jupyter Notebook` (주피터 노트북) 사용 경험 (선택 사항이지만 권장됨)
*   **`Panel` 설치:** 터미널 또는 노트북에서 `pip install panel`을 실행하세요.

## 시작하기

`Panel`의 핵심 기능을 보여주는 간단한 예제를 살펴보겠습니다. 제목, 부제목, 위젯 및 레이아웃 생성에 대해 다룰 것입니다.

### 1단계: Panel 설정

먼저 `Panel`을 임포트하고 `Jupyter` 확장 기능을 활성화합니다 (노트북을 사용하는 경우).

```python
import panel as pn
pn.extension()  # Enables Panel in Jupyter
```

이는 `Panel`이 노트북 내에서 또는 독립 실행형 (standalone) 앱으로 컴포넌트를 렌더링하도록 설정합니다.

### 2단계: 제목과 부제목 추가하기

`Panel`을 사용하면 `Markdown` 또는 `HTML`을 이용해 제목이나 부제목과 같은 텍스트 요소를 쉽게 추가할 수 있습니다.

**예제: 제목과 부제목 만들기**
```python
import panel as pn
pn.extension()

# Create a title and subtitle using Markdown
title = pn.pane.Markdown("# My First Panel Dashboard")
subtitle = pn.pane.Markdown("## Exploring Data with Interactive Widgets")
# Display them
pn.Column(title, subtitle).servable()
```

**설명:**
*   `pn.pane.Markdown`: `Markdown` 텍스트를 렌더링합니다. `#`은 제목에, `##`은 부제목에 사용합니다.
*   `pn.Column`: 컴포넌트를 수직으로 배열합니다.
*   `.servable()`: 레이아웃을 노트북이나 웹 앱에 표시할 준비가 된 상태로 만듭니다.

**결과:** `Jupyter Notebook`에서 이 코드를 실행하면 굵은 글씨의 제목("My First Panel Dashboard")과 그 아래에 더 작은 부제목이 표시됩니다.

*이미지를 전체 크기로 보려면 Enter를 누르거나 클릭하세요.*

### 3단계: 인터랙티브 위젯 추가하기

`Panel`은 슬라이더, 버튼, 드롭다운과 같은 인터랙티브 위젯에서 탁월한 성능을 발휘합니다. 숫자를 제어하는 간단한 슬라이더를 만들어 보겠습니다.

**예제: 슬라이더 위젯**
```python
import panel as pn
pn.extension()

# Create title and subtitle
title = pn.pane.Markdown("# Interactive Slider Demo")
subtitle = pn.pane.Markdown("## Adjust the slider to see the value change")

# Create a slider widget
slider = pn.widgets.IntSlider(name="Value", start=0, end=100, value=50)

# Create a function to display the slider's value
def display_value(value):
    return pn.pane.Markdown(f"Current Value: **{value}**")
# Bind the slider value to the display function
interactive_output = pn.bind(display_value, slider)
# Arrange components in a layout
layout = pn.Column(title, subtitle, slider, interactive_output)
layout.servable()
```

**설명:**
*   `pn.widgets.IntSlider`: `start`와 `end` 사이의 정수 값을 위한 슬라이더를 생성합니다.
*   `pn.bind`: 슬라이더의 값을 동적으로 업데이트되는 함수(`display_value`)에 연결합니다.
*   레이아웃은 제목, 부제목, 슬라이더 및 출력 텍스트를 수직으로 쌓습니다.

**결과:** 제목, 부제목, 슬라이더, 그리고 슬라이더를 움직일 때마다 업데이트되는 텍스트(예: "Current Value: 45")가 표시됩니다.

*이미지를 전체 크기로 보려면 Enter를 누르거나 클릭하세요.*

### 4단계: Panel로 데이터 시각화하기

`Panel`은 `Matplotlib`와 같은 플로팅 라이브러리와 통합됩니다. 슬라이더가 사인파 (sine wave)의 주파수를 제어하는 인터랙티브 플롯을 만들어 보겠습니다.

**예제: 인터랙티브 사인파 플롯**
```python
import panel as pn
import numpy as np
import matplotlib.pyplot as plt
pn.extension('plotly')  # Enable Plotly support

# Create title and subtitle
title = pn.pane.Markdown("# Interactive Sine Wave")
subtitle = pn.pane.Markdown("## Adjust the frequency of the wave")

# Create a slider for frequency
freq_slider = pn.widgets.FloatSlider(name="Frequency", start=1, end=10, value=2)

# Function to create a sine wave plot
def plot_sine(frequency):
    x = np.linspace(0, 4 * np.pi, 100)
    y = np.sin(frequency * x)
    fig, ax = plt.subplots()
    ax.plot(x, y)
    ax.set_title(f"Sine Wave (Frequency = {frequency})")
    plt.close(fig)  # Close the figure to avoid duplicate displays
    return fig

# Bind the slider to the plot function
interactive_plot = pn.bind(plot_sine, frequency=freq_slider)

# Create the layout
layout = pn.Column(title, subtitle, freq_slider, interactive_plot)
layout.servable()
```

**설명:**
*   `pn.widgets.FloatSlider`: 부동 소수점 (floating-point) 값을 위한 슬라이더입니다.
*   `plot_sine`: 슬라이더의 주파수를 기반으로 사인파의 `Matplotlib` 플롯을 생성합니다.
*   `pn.bind`: 슬라이더 값이 변경될 때마다 플롯을 업데이트합니다.
*   `pn.extension('plotly')`: `Matplotlib`/`Plotly` 렌더링과의 호환성을 보장합니다.

**결과:** 제목, 부제목, 슬라이더, 그리고 주파수를 조정함에 따라 업데이트되는 사인파 플롯이 표시됩니다.

*이미지를 전체 크기로 보려면 Enter를 누르거나 클릭하세요.*

### 5단계: 간단한 대시보드 만들기

드롭다운 (dropdown)과 플롯을 사용하여 여러 요소를 작은 대시보드로 결합해 보겠습니다.

**예제: 인터랙티브 데이터 대시보드**
```python
import panel as pn
import pandas as pd
import plotly.express as px
pn.extension('plotly')

# Create title and subtitle
title = pn.pane.Markdown("# Simple Data Dashboard")
subtitle = pn.pane.Markdown("## Select a dataset to visualize")

# Sample data
df = pd.DataFrame({
    'x': range(10),
    'y1': [1, 2, 4, 3, 5, 6, 4, 7, 8, 9],
    'y2': [2, 4, 1, 5, 3, 7, 2, 8, 6, 9]
})

# Dropdown to select a column
column_select = pn.widgets.Select(name="Select Column", options=['y1', 'y2'], value='y1')

# Function to plot selected column
def plot_data(column):
    fig = px.line(df, x='x', y=column, title=f"Plot of {column}")
    return fig

# Bind dropdown to plot function
interactive_plot = pn.bind(plot_data, column=column_select)

# Create the layout
layout = pn.Column(title, subtitle, column_select, interactive_plot)
layout.servable()
```

**설명:**
*   `pn.widgets.Select`: `y1`과 `y2` 중 하나를 선택하는 드롭다운 (dropdown) 메뉴입니다.
*   `plot_data`: 선택된 열을 기반으로 `Plotly Express`를 사용하여 라인 플롯을 생성합니다.
*   레이아웃은 제목, 부제목, 드롭다운 및 플롯을 결합합니다.

**결과:** 제목, 부제목, 드롭다운 메뉴, 그리고 선택된 열에 따라 업데이트되는 라인 플롯이 있는 대시보드가 표시됩니다.

### 6단계: 웹 앱으로 실행하기

```bash
panel serve your_script.py --show
```

`your_script.py`를 `Panel` 코드가 포함된 파이썬 파일 이름으로 바꾸세요. `--show` 플래그는 브라우저에서 앱을 엽니다.

## `Panel`의 주요 기능
*   **위젯 (Widgets):** 슬라이더, 버튼, 드롭다운, 텍스트 입력 등
*   **레이아웃 (Layouts):** `pn.Column`, `pn.Row`, 또는 `pn.Grid`를 사용하여 컴포넌트를 구성합니다.
*   **통합 (Integration):** `Matplotlib`, `Plotly`, `Bokeh` 등과 연동됩니다.
*   **배포 (Deployment):** 로컬에서 실행하거나, 서버에 배포하거나, `Binder`를 통해 공유할 수 있습니다.
