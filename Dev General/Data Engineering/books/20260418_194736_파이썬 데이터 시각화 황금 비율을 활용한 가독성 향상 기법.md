# 파이썬 데이터 시각화: 황금 비율을 활용한 가독성 향상 기법

대부분의 사람들은 보기 좋지 않은 차트가 나쁜 데이터 때문이라고 생각하지만, 항상 그렇지는 않습니다. 종종 차트의 모양, 즉 레이아웃(layout)이 문제인 경우가 많습니다.

파이썬(Python)에서 차트를 생성할 때, 라이브러리(library)는 기본 크기를 선택합니다. 때로는 정사각형으로, 때로는 가로로 길게, 때로는 세로로 길게 설정됩니다. 이러한 직사각형 모양은 차트의 가독성에 직접적인 영향을 미칩니다. 그리고 다른 디자인 목표를 희생하지 않는 한, 일관적으로 잘 작동하는 간단한 모양이 존재합니다.

디자이너들은 수백 년 동안 이를 사용해 왔는데, 바로 황금 비율(Golden Ratio)이라고 불립니다. 이는 파이(Phi, φ ≈ 1.618)라고도 불리는 간단한 숫자입니다.

황금 비율은 데이터 시각화에 적용하기 매우 쉽습니다. 실제 데이터셋인 세계 평화 지수(Global Peace Index)를 사용하여 그 방법을 보여드리겠습니다.

## 데이터셋

세계 평화 지수(Global Peace Index, GPI) 데이터셋은 2008년부터 2022년까지 각국의 평화 점수를 포함합니다. GPI에서 점수가 낮을수록 해당 국가가 더 평화롭다는 것을 의미합니다. 예를 들어, 현재 아이슬란드(Iceland)는 점수가 1에 가장 가깝기 때문에 가장 평화로운 국가입니다. 데이터셋을 불러오는 코드는 다음과 같습니다:

```python
import pandas as pd

df = pd.read_csv("global_peace_index.csv")
df.head()
```

이 데이터셋은 각 국가별로 한 행을, 각 연도별로 열을 제공합니다. 이제 이를 시각화할 수 있습니다. 데이터를 준비했으니, 이제 세 가지 예시를 보여드리겠습니다. 각 예시는 다음 두 가지를 포함합니다:

*   정량적으로는 정확하지만 황금 비율에 맞지 않는 표준 차트;
*   황금 비율이 적용된 동일한 차트.

어떤 차트가 더 나은지 직접 판단해 보시기 바랍니다.

## 차트 1: 국가별 시계열 플롯(Time Series Plot)

황금 비율 탐색을 위해 한 국가의 시계열 차트(time-series chart)를 그려보겠습니다.

### 표준 차트 표시

일반적인 기본 그림 크기로 설정된 코드는 다음과 같습니다:

```python
import matplotlib.pyplot as plt

country = "United Kingdom"
years = list(range(2008, 2023))
values = df[df["Country"] == country][[str(y) for y in years]].values.flatten()
plt.figure(figsize=(6,6))
plt.plot(years, values)
plt.title("United Kingdom Peace Score")
plt.show()
```

이것이 기본 차트입니다. 이 차트는 기술적으로는 정확하지만, 조금 답답하고, 상자 같고, 어딘가 불균형하게 느껴지지 않나요? 이는 정사각형 모양 때문에 발생합니다.

### 황금 비율이 적용된 동일 차트

이제 황금 비율을 적용하여 모양만 바꿔보겠습니다:

```python
phi = 1.618

plt.figure(figsize=(6, 6/phi))
plt.plot(years, values)
plt.title("United Kingdom Peace Score")

plt.show()
```

그 결과는 다음과 같습니다: 와우, 선을 훨씬 쉽게 따라갈 수 있고 타임라인(timeline)이 훨씬 명확해졌습니다. 이것이 바로 ‘작동하는’ 황금 비율입니다. 레이아웃(layout)이 중요한 이유가 여기에 있습니다.

## 차트 2: 모든 국가를 비교하는 산점도(Scatterplot)

이제 2008년과 2022년 사이의 평화 점수를 비교해 보겠습니다. 각 점은 한 국가를 나타냅니다. 먼저, 기본 레이아웃(layout)을 생성해 봅시다:

```python
plt.figure(figsize=(6,6))
plt.scatter(df["2008"], df["2022"])
plt.title("Peace Score Comparison")
plt.show()
```

결과: 다시 공간이 답답해 보입니다. 틀린 것은 아니지만, 글의 흐름에 제대로 맞지 않는 것 같습니다.

### 황금 비율이 적용된 동일 차트

이제 황금 비율을 적용하여 모양만 바꿔보겠습니다:

```python
plt.figure(figsize=(10, 10/phi))
plt.scatter(df["2008"], df["2022"])
plt.title("Peace Score Comparison")
plt.show()
```

이 아주 미세한 변화는 다음과 같은 결과를 만들어냅니다: 이 버전은 시각적으로 스캔하기가 더 쉽습니다. 클러스터(cluster)를 더 쉽게 구분할 수 있으며, 이상치(outlier)가 훨씬 명확하게 드러납니다. 차트가 더 숨 쉬는 것 같습니다.

## 차트 3: 상위 국가 순위를 매기는 막대형 차트(Bar Chart)

2022년 가장 평화롭지 않은 국가들을 순위 매겨보겠습니다. 먼저 기본 세로 막대형 레이아웃(layout)부터 시작하겠습니다:

```python
worst = df_years[["Country", rank_year]].dropna().sort_values(rank_year, ascending=False).head(15)
worst = worst.iloc[::-1]  # reverse for nicer top-to-bottom reading

# 1) Common
plt.figure(figsize=(10, 10))  # phi-wide-ish
plt.bar(worst["Country"], worst[rank_year], color='green')
plt.title(f"15 least peaceful countries ({rank_year}) — default")
plt.xlabel("GPI score")
plt.grid(True, axis="x", alpha=0.2)
plt.tight_layout()
plt.show()
```

매우 보기 싫고 답답한 결과는 다음과 같습니다: 네, 정말 끔찍합니다. X축을 정리해야 하며, 이 차트는 글에서 너무 많은 세로 공간을 차지하여 독자가 스크롤(scroll)을 훨씬 더 많이 해야 합니다. 좋지 않습니다.

### 황금 비율이 적용된 (거의) 동일 차트

이제 황금 비율을 적용하여 모양만 바꿔보겠습니다(여기에 `barh()` 함수를 사용하여 차트를 가로로 바꾸는 추가적인 이점도 있습니다):

```python
plt.figure(figsize=(10, 10/phi))  # phi-wide-ish
plt.barh(worst["Country"], worst[rank_year], color='green')
plt.title(f"15 least peaceful countries ({rank_year}) — phi-wide")
plt.xlabel("GPI score")
plt.grid(True, axis="x", alpha=0.2)
plt.tight_layout()
plt.show()
```

그리고 놀랍도록 개선된 결과는 다음과 같습니다: 이 차트는 가로 막대형 차트로 만들어졌으며 황금 비율이 적용되었습니다. 훨씬 더 보기 좋습니다. 정말 비교할 수 없을 정도로 개선되었습니다. 차트가 더 명확하고, 덜 혼잡하며, 더 전문적으로 느껴집니다.

## 이것이 작동하는 이유

차트는 시각적인 객체이며, 우리 뇌는 모양과 공간을 자동으로 해석합니다. 차트가 너무 정사각형이면 데이터가 압축되어 보이고, 너무 길면 추세(trend)가 과장되어 보입니다. 황금 비율은 균형 잡힌 비율을 제공하여 차트의 가독성을 높여줍니다. 데이터 자체를 변경하지 않고도 명확성을 향상시킵니다.

## 따라야 할 간단한 규칙

이제부터 차트와 그래프를 만들 때, 임의의 크기를 사용하는 대신 다음을 사용해 보세요:

```python
phi = 1.618
plt.figure(figsize=(10, 10/phi))
```

이 방법은 대부분의 차트를 즉시 개선합니다.

**참고:** 황금 비율이 디자인의 다른 측면을 훼손하는 경우에는 구현하지 않는 것이 좋습니다. 예를 들어, 차트에서 X축과 Y축이 동일하게 중요하다면 정사각형 차트가 가장 좋습니다.

## 요약

대부분의 초보자들은 데이터와 차트 유형에 집중하지만, 레이아웃(layout) 또한 그만큼 중요할 수 있습니다. 좋지 않은 레이아웃은 비전문적인 분석이라는 인상을 줄 수 있습니다.

황금 비율을 사용하면(다른 디자인 요소를 희생하지 않는 한) 균형 잡히고 가독성 높은 차트를 만들 수 있습니다. 따라서 종횡비(aspect ratio)를 나중에 생각하는 요소로 여기지 말고, 스토리텔링(storytelling)의 일부로 다루십시오.

이는 데이터 스토리텔링을 즉시 개선할 수 있는 가장 쉬운 방법 중 하나입니다.
