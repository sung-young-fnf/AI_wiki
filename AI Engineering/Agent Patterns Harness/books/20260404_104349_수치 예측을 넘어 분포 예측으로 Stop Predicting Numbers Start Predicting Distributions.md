# 수치 예측을 넘어 분포 예측으로 (Stop Predicting Numbers. Start Predicting Distributions.)

모델이 "내일 매출은 4,200개일 것"이라고 예측했을 때, 그 의미는 무엇일까요? 모델이 얼마나 확신하고 있나요? 3,000개나 6,000개가 될 가능성도 동일하게 존재할까요? 분포가 대칭적일까요, 아니면 한쪽으로 치우친 위험 요소(long tail of risk)가 있을까요?

점 예측(point prediction)은 이러한 정보를 전혀 제공하지 않습니다. 그럼에도 불구하고 실제 운영 환경의 대부분의 회귀 모델은 샘플당 단 하나의 숫자만을 출력합니다.

이것은 매우 중요한 문제입니다. 재고 수준을 설정하거나 보험료를 책정하거나 자원을 할당할 때, 예측값 주변의 불확실성이 어떤 형태를 띠고 있는지가 예측값 그 자체보다 더 가치 있는 경우가 많기 때문입니다. 평균값이 같더라도 분포가 다봉형(bimodal, 두 개의 가능성이 높은 결과)인 경우는 좁은 단봉형(unimodal) 분포일 때와 근본적으로 다른 결정을 내려야 합니다.

**DistributionRegressor**는 단일 LightGBM 모델을 사용하여 전체 조건부 확률 분포 $p(y|x)$를 예측하는 Python 패키지입니다. 비모수적(nonparametric)이며 속도가 빠르고, 사전에 분포에 대한 가정을 요구하지 않습니다.

### 모수적 분포 회귀의 문제점

분포 회귀에서 가장 잘 알려진 접근 방식은 **NGBoost**입니다. 이는 자연 경사 부스팅(natural gradient boosting)을 사용하여 정규 분포, 로그 정규 분포 등 선택된 분포 가족(distribution family)의 파라미터를 학습합니다.

NGBoost는 우아한 방식이지만 근본적인 제약이 있습니다. 분포 가족을 사전에 선택해야 한다는 점입니다. 만약 정규 분포를 선택했는데 실제 조건부 분포가 다봉형이라면, 아무리 많은 학습 데이터를 투입해도 모델은 구조적으로 실제 분포를 표현할 수 없으므로 모든 예측이 틀리게 됩니다.

현실 세계의 조건부 분포는 다음과 같은 경우가 많습니다:
*   **다봉형(Multimodal):** 통근 시간(교통 체증 유무), 주택 가격(지역 혼합) 등
*   **영과잉(Zero-inflated):** 소매 판매(매장 개점/폐점), 보험 청구 등
*   **심한 왜도와 복잡한 꼬리:** 의료비, 금융 수익률 등

### 새로운 접근 방식: 누적 분포 함수(CDF) 직접 학습

DistributionRegressor는 분포 파라미터를 맞추는 대신, 조건부 누적 분포 함수(Cumulative Distribution Function, CDF)를 직접 학습함으로써 모수적 가정을 우회합니다.

핵심 아이디어는 이를 이진 분류(binary classification) 문제로 전환하는 것입니다. 특정 임계값(threshold) $\tau$와 학습 샘플 $(x_i, y_i)$가 있을 때, 우리는 $y_i$가 $\tau$보다 낮은지 여부를 정확히 알 수 있습니다. 이는 단순한 이진 레이블이며, LightGBM은 이진 분류에 매우 뛰어난 성능을 보입니다.

**작동 방식 4단계:**

1.  **그리드 생성:** 타겟 범위를 $K$개의 임계값 점 $\tau_1, \tau_2, \dots, \tau_K$로 이산화합니다.
2.  **이진 타겟 생성:** 각 학습 샘플과 각 임계값에 대해 $z_{ij} = \mathbb{1}\{y_i \le \tau_j\}$를 계산합니다.
3.  **데이터셋 확장:** 모든 샘플과 모든 임계값 점을 교차 결합합니다. $N$개의 샘플 데이터셋은 원본 특성에 임계값을 추가 특성으로 포함한 $N \times K$ 행의 데이터셋이 됩니다.
4.  **모델 학습:** 단일 LightGBM 모델을 교차 엔트로피(cross-entropy) 손실 함수로 학습시키되, 임계값 특성에 대해 **단조 증가 제약 조건(monotone increasing constraint)**을 부여합니다. 이 제약 조건은 모델이 유효한 CDF(임계값에 따라 감소하지 않음)를 출력하도록 보장하는 핵심 요소입니다.

추론 시 모델은 모든 그리드 점에 대해 $F(\tau|x)$를 예측하고, 이 CDF의 차분(differencing)을 통해 확률 질량 함수(PMF)를 복원합니다.

### 이 방식이 효과적인 이유

*   **밀도 높은 신호:** 단일 샘플 $(x_i, y_i)$가 $K$개의 학습 예시를 생성하므로 학습 데이터가 풍부해집니다.
*   **내재된 단조성:** LightGBM의 단조 제약 조건을 통해 별도의 후처리 없이도 항상 유효한 확률 분포를 얻습니다.
*   **교정된 확률:** 로지스틱 손실(Logistic loss)은 모델이 잘 교정된(calibrated) 확률을 출력하도록 유도하며, 이는 CDF 값에 최적입니다.
*   **자유로운 분포 형태:** 다봉형, 왜도, 영과잉 분포 등 데이터가 요구하는 어떤 형태의 분포도 학습할 수 있습니다.

### 코드 예시

```python
from distribution_regressor import DistributionRegressor

# 모델 학습
model = DistributionRegressor(
    n_bins=100,
    n_estimators=1000,
    learning_rate=0.01,
)
model.fit(X_train, y_train)

# 점 예측
y_mean = model.predict(X_test)          # 기댓값 (평균)
y_mode = model.predict_mode(X_test)     # 최빈값 (밀도 피크)

# 불확실성 정량화
intervals = model.predict_interval(X_test, confidence=0.9) # 신뢰 구간
std = model.predict_std(X_test) # 표준 편차

# 전체 분포 및 확률 밀도 평가
grids, dists, _ = model.predict_distribution(X_test)
nll = model.nll(X_test, y_test)  # 음의 로그 가능도 (NLL)
```

### 실제 사례: 소매 판매 및 주택 가격 예측

**소매 판매 예측 (Rossmann 데이터셋):**
모델은 상황에 따라 유연하게 변하는 예측 구간을 생성합니다. 프로모션이 있는 날은 불확실성 밴드가 넓어지고, 매장이 문을 닫는 일요일은 0으로 수렴합니다. 이는 가우시안 분포를 강제로 적용한 결과가 아니라 데이터 자체의 특성을 학습한 결과입니다.

**주택 가격 예측 (California Housing 데이터셋):**
NGBoost와 비교했을 때, DistributionRegressor는 훨씬 더 우수한 성능을 보였습니다. NGBoost의 정규 분포 가정은 모든 예측을 대칭적인 종 모양으로 강제하는 반면, DistributionRegressor는 실제 데이터의 복잡한 다봉형 밀도를 정확히 포착합니다.

### 메모리 고려 사항

데이터셋이 $N \times K$로 확장되기 때문에 메모리 사용량이 주요 고려 사항입니다. 이를 해결하기 위해 **몬테카를로 학습 모드(Monte Carlo training mode)**를 제공합니다. 학습 샘플당 그리드 점의 일부만 샘플링하여 메모리 사용량을 5~20배 줄이면서도 전체 그리드를 사용했을 때와 유사한 성능을 낼 수 있습니다.

### 핵심 설계 결정 사항

1.  **단일 모델 사용:** 임계값별로 별도의 분류기를 만드는 대신, 임계값을 특성으로 활용하여 정보를 공유하고 파라미터 수를 줄였습니다.
2.  **교차 엔트로피 손실:** MSE보다 확률 교정에 훨씬 유리합니다.
3.  **단조 제약 조건:** 후처리 수정보다 훨씬 매끄럽고 안정적인 CDF를 생성합니다.
4.  **출력 평활화(Smoothing):** 학습 후 예측된 CDF에 가우시안 평활화를 선택적으로 적용할 수 있습니다.

### 언제 사용해야 하는가?

*   단순한 점 예측이 아닌 전체 조건부 분포가 필요할 때
*   실제 분포의 형태를 알 수 없거나 복잡할 때 (다봉형, 영과잉 등)
*   분포에 대한 가정이 없는 비모수적 접근 방식을 원할 때
*   데이터가 정형(tabular) 데이터이며, scikit-learn과 호환되는 워크플로우를 원할 때

시퀀스나 이미지와 같은 구조화된 출력의 분포가 필요하거나, 데이터의 차원이 매우 높은 경우(신경망 밀도 추정기 권장), 또는 특정 모수적 가족에 대한 강력한 사전 지식이 있는 경우(NGBoost가 더 유리함)가 아니라면 DistributionRegressor는 매우 강력한 도구가 될 것입니다.
